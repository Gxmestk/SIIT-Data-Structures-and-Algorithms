# SIIT-Data-Structures-and-Algorithms

Practicing Data Structures and Algorithms with Java

## Course Overview

This repository contains lab exercises for the Data Structures and Algorithms course at SIIT (Sirindhorn International Institute of Technology).

## Labs

| Lab | Topic | Description |
|-----|-------|-------------|
| Lab00 | Intro to Java and Looping | Basic Java syntax and loop constructs |
| Lab01 | Array and Recursion | Array operations and recursive programming |
| Lab02 | Sorting and Hashing | Sorting algorithms and hash table implementation |
| Lab03 | Singly Linked List | Linked list data structure implementation |
| Lab04 | Stack and Applications | Stack data structure and its applications |
| Lab05 | Queue and Applications | Queue data structure and its applications |
| Lab06 | Binary Tree | Binary tree implementation and traversal |
| Lab07 | Binary Search Tree | BST operations: insertion, deletion, search |
| Lab08 | Heap | Heap data structure and heap operations |
| Lab09 | Graph | Graph representation and algorithms |

## Project Structure

```
LabXX/
├── src/              # Source code (.java files)
├── bin/              # Compiled class files
└── Lab Instruction/  # Lab PDFs and starter code
```

## How to Run

Each lab is an Eclipse project. You can:

1. Open in Eclipse: Import as existing project
2. Compile manually:
   ```bash
   cd LabXX/src
   javac *.java
   java TestClassName
   ```
